<h1>Click-Through Propensity Modeling for Online Ads</h1>

<h2>Project overview</h2>
<p>
A marketing company collected user data such as age, location, and online activity to
understand which users are likely to click on digital advertisements. This project uses
logistic regression to predict ad clicks and support targeted marketing strategies.
</p>

<h2>Business problem</h2>
<p>
The company wanted to:
</p>
<ul>
  <li><b>Increase ad efficiency</b> by showing ads to users more likely to click.</li>
  <li><b>Reduce wasted impressions</b> on uninterested users.</li>
  <li><b>Understand which user attributes</b> drive engagement with ads.</li>
</ul>

<h2>Objectives</h2>
<ul>
  <li><b>Build a logistic regression model</b> to predict whether a user will click an ad.</li>
  <li><b>Evaluate model performance</b> using classification metrics and ROC-AUC.</li>
  <li><b>Interpret feature effects</b> on click probability.</li>
</ul>

<h2>Data and features</h2>
<p>
The dataset includes:
</p>
<ul>
  <li><b>User profile:</b> age, gender, location.</li>
  <li><b>Online behavior:</b> time spent on site, daily internet usage.</li>
  <li><b>Ad context:</b> timestamp, ad placement (if available).</li>
  <li><b>Target variable:</b> clicked vs not clicked.</li>
</ul>

<h2>Methodology</h2>
<ul>
  <li><b>Data preprocessing:</b> cleaning, encoding categorical variables, scaling numeric features.</li>
  <li><b>EDA:</b> univariate and bivariate analysis of click behavior.</li>
  <li><b>Modeling:</b> logistic regression on scaled data.</li>
  <li><b>Evaluation:</b> confusion matrix, precision, recall, ROC curve, and AUC.</li>
</ul>

<h2>Key results</h2>
<ul>
  <li><b>Click probability scores</b> for each user.</li>
  <li><b>Insights into high-engagement segments</b> based on age, usage, and location.</li>
  <li><b>Model performance summary</b> for marketing decision-makers.</li>
</ul>

<h2>Business impact</h2>
<ul>
  <li><b>Better targeting:</b> focus campaigns on high-propensity users.</li>
  <li><b>Higher ROI:</b> improved click-through rates and conversion potential.</li>
  <li><b>Audience insights:</b> clearer understanding of who responds to ads.</li>
</ul>
